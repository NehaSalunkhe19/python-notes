

## 2. if-else Statement

The if-else statement is used when we want to execute one block if the condition is true and another block if it is false.

Syntax
if condition:
    statement(s)
else:
    statement(s)
    

Example
number = 5

if number % 2 == 0:
    print("Even number")
else:
    print("Odd number")



Output
Odd number
