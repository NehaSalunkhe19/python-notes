3. if-elif-else Statement

The elif (else if) statement is used to check multiple conditions.

Syntax
if condition1:
    statement(s)
elif condition2:
    statement(s)
else:
    statement(s)


Example
marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 60:
    print("Grade B")
else:
    print("Grade C")


Output
Grade B
